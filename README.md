# Kickr

## Get Started

1. `npm install`
1. `touch .env`
1. edit `.env` file
1. `npm start`
1. go to <http://localhost:4390>

## .env settings

```bash
CLIENT_ID=<CLIENT_ID>
CLIENT_SECRET=<CLIENT_SECRET>
WEBHOOK_URL=<WEBHOOK_URL>
PORT=<PORT>
SLACK_VERIFICATION_TOKEN=<SLACK_VERIFICATION_TOKEN>
SLACK_ACCESS_TOKEN=<SLACK_ACCESS_TOKEN>
```
